# warnock
Warnock algorithm and drawing 3d objects in 2d space.

# Required
OpenGL and SDL library.

# Building

    $ make
    $ ./warnock

# Results
![alt tag](https://raw.github.com/oshogbo/warnock/master/result.png)

# Usage
    PgUP/PgDown/Up/Down/Left/Right - move
    a/d/w/s - rotate
